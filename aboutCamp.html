About the Camp

This is not your ordinary coding camp. It is an interactive camp in which by the end of the 5 hours, you will have coded an entire project that you make, from the the idea to the design to the actual code is done by you. And the best part? It's completly free and there is someone to help you along the whole way. You coding skills will grow as you learn critcal skills as you code an entire project from start to finish Whether you are beginner looking to start block-coding or text-coding or a professional looking to hone your skill, this free class is for everyone. All ages are welcome.
